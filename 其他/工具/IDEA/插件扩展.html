插件扩展

发表于2023-12-17|更新于2023-12-17|其他工具IDEA
|字数总计:495|阅读时长:1分钟|阅读量:
  1. Key Promoter X
    快捷键神器，功能如下
    1. 非快捷键方式做有快捷键操作时，提示对应快捷键
    2. 无快捷键键操作，但可以设置时，提示设置
    3. 通知窗口中，找到某个快捷键的通知，key binding可以快捷跳转该快捷键的快捷键设置
  2. JRebel and XRebel
    收费插件，改sql不用重启后端的插件
  3. Chinese (Simplified) Language Pack / 中文语言包
    jb官方的汉化插件，菜单、设置等的汉化
  4. Translation
    翻译插件，支持编辑区、控制台翻译，可惜设置中的部分没办法翻译
  5. Rainbow Brackets
    收费插件，美化插件，可以用不同颜色标识不同缩进和层级，不同颜色的前端标签等，有些付费功能，不过免费的就够了
  6. Grep Console
    控制台美化插件，具体我也还摸不清楚。。
  7. MyBatisCodeHelperPro
    收费插件（介意可以选择 Mabatis X ，也支持很多功能），不过最主要使用的功能还是使得mapper.java和mapper.xml互相跳转，其他功能也很有用，比如根据方法名生成sql、根据表生成一整套模板（但是不知道咋地，无论这个还是Mabatis X 都不行。。）
  8. MyBatis Log Free
    可以开一个窗口，监听到某个控制台打印的sql，而且是赋了参数的
  9. Extra ToolWindow Colorful Icons
    图标美化插件，使得图标变成彩色
  10. GitToolBox
    收费插件，不过其他功能也没细看。只用着免费版的显示贡献人功能，也就是git blame
  11. Alibaba Java Coding Guidelines
    阿里巴巴规约插件，但是好像已经比较久不更新了。。而且听说比较容易卡顿，我也没经常用过，或许关闭实时扫描会好很多
  12. CamelCase
    快速在小驼峰、大驼峰、下划线之类的单词风格转换
  13. CodeGlance Pro
    显示代码缩略图在编辑区（vscode基本都有这个）
