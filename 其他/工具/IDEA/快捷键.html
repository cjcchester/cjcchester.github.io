快捷键

发表于2023-12-16|更新于2023-12-16|其他工具IDEA
|字数总计:719|阅读时长:2分钟|阅读量:

前言

快捷键的本质是什么？是一个操作，不管是快捷键还是放在菜单某个位置用鼠标点都只是功能的操作方式。

所以，具体是哪个快捷键并不重要。重要的是，你知道你在使用的东西，这里是idea，给你提供了这样的功能。在你需要的时候，你可以想起来有这个功能可以快速的完成需要的操作，这就够了。

JB的快捷键应该是通用的，然后也可以选择风格，各大IDE基本上快捷键风格都可以互选。我常用IDEA的，往往是原设。
这里说一下，vscode切换idea快捷键的办法。是一个插件——IntelliJ IDEA Keybindings

然后要说的是，下面只是一些本人常用的快捷键，还在完善中，用到想起来会记一下，不会很全面。

正文

  1. Ctrl + Alt + L
    格式化代码

  2. Ctrl + Alt + 鼠标左键 或者 Ctrl + Alt + B
    从controller跳转到serviceImpl、从impl跳转到mapper xml（忘了有无插件的作用，如果有，那就是mybatisX）

  3. Ctrl + B
    转到声明或引用

  4. Ctrl + Shift + F
    项目文件全文搜索。右上角有个文件筛选，我常用的：

  5. Alt + Enter
    万能快捷键。最常用的用法包括
    手动导入包、类；
    在service层新增了一个方法，在方法上使用，可以拉取实现到对应的impl。

  6. Ctrl + Alt + O
    手动移除未使用包

  7. Ctrl + [ Ctrl + ]
    快速跳转方法{}大括号的位置，可以配置方法分隔横线使用

  8. Alt + ← Alt + →
    上一个/下一个 标签页

  9. Ctrl + Shift + Enter
    代码结尾补全。如一句代码结尾的;、if(xx)补全大括号；
    进阶：非常好用，能直接补全代码，不用手动跳出括号，还会把该句格式化一下。唯一的缺点是a==b.if生成的if再用它是格式化不了了，不过手动选中if语句，按下快捷键，也可以格式化。

  10. Shift 双击
    全局搜索，不限于项目文件，还包括配置、类库等等文件的内容

  11. Ctrl + Shift + N
    快速找到文件
    进阶：支持/和/和空格（但不支持目录逆序），支持后缀，比如有目录 abcd/abc.java，你用ab / java 或 ab \ java是可以搜到的！

  12. Ctrl + Shift + Alt + N
    快速找到文件中的方法，只支持.

  13. Ctrl + Shift + I
    预览（类、方法）（预览里无法再嵌套预览了）

  14. Ctrl + Alt + H
    看方法在哪些地方被调用
    进阶：会弹出一个窗口，双击跳转源代码位置，也可以用 Ctrl + Shift + I 预览！

  15. ctrl + F12
    查看当前类的所有方法，有时候，你要快速去查看某个类有没有某个方法可以用上。
